# SimpleTimer

Простой таймер для windows, сделанный как учебный проект.
Написан на языке Python.
Настройка значений производится колесом мыши.
Поддерживает одновременное включение нескольких таймеров.
По завершению приходит простенькое уведомление и сопровождается мелодией до отключения или конца мелодии.

Доступна возможность выбирать между темной и светлой темой.
